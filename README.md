黒歴史クリーナーはツイート一括削除するためのサービスです。
簡単な操作でアカウントに登録されているツイートを一括で削除できます。

## Features

- [簡単操作] Twitterでログインして自分のアカウント名を入力するだけ！あとは全て自動で削除されます。
- [ステータス確認] 処理過程をステータスとして確認できます。
- [処理の中止] 任意のタイミングで削除処理を取りやめることができます。
